Домашнее задание – тема ввод-вывод из файлов (продолжение предыдущих ДЗ)

1. Создать структуру ВИДЕОФИЛЬМ со следующими полями:
Название фильма
Режиссер
Жанр
Рейтинг популярности
Цена диска
2. Реализовать меню при помощи массива указателей на функции, где есть следующие пункты:
Поиск по названию
Поиск по жанру
Поиск по режиссеру
Самый популярный фильм в жанре
Показ всех записей
Добавление фильма
2. Вынести в отдельные файлы следующую функциональность:
Определение структуры ВИДЕОФИЛЬМ
Функцию main
Функцию реализации меню
Функции работы с фильмами (все операции из пункта 2 выше)
3. Добавить сохранение в файл новых видеофильмов
4. Добавить при старте программы чтение всех видеофильмов введенных ранее из файла
5. Создать отдельный файл для работы с функциями сохранения/чтения в/из файла
